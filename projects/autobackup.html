Automatic Back Up



Description:

This Python desktop application provides a user-friendly interface built with Tkinter for selecting multiple source folders and a destination folder to perform automated local backups. It detects changes in source folders and creates timestamped backup copies.

The application supports optional uploads to Google Drive using PyDrive, maintains a log of the last backup time, and allows users to configure settings and schedule backups. It uses Windows Task Scheduler to set scheduled times and enable automatic backups without user intervention.
