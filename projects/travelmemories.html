Travel Memories



Live Demo:

  • https://finalmedp.glitch.me/


    • Description:

    This project was developed by me and a teammate as a digital travel journal where users can save both past trips and future dream destinations. Users are able to write entries with their thoughts, add tags, select dates, and later edit or delete their posts.

    The design was inspired by the idea of a postbox, where each memory acts like a letter randomly added to a collection. Posts are displayed in a scattered, postcard-like layout to evoke a sense of nostalgia and personal storytelling.


    Framework:

  • Node.js
  • Express.js
  • MongoDB
  • Handlebars (hbs) - View Engine


    • Features:

  • Users can create new post, edit their post later, delete post
  • Users can drag their posts
  • Instructions drop down button to let users understand how to use the app
  • Filter posts based on their tags
